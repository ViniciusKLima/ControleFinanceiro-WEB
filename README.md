# 💰 Controle Financeiro - Landing Page

Landing page desenvolvida para apresentar um aplicativo de controle financeiro.

## 🚀 Tecnologias
- React
- Vite
- Bootstrap

## 🎯 Objetivo
Apresentar de forma clara e atrativa as funcionalidades do app, focando na experiência do usuário e conversão.

## 📱 Funcionalidades do site
- Seção Hero
- Problemas do usuário
- Benefícios
- Funcionalidades
- Comparação (com vs sem app)
- CTA
- Contato
